# Dissertation: Gender and Climate Discourse in the European Parliament

This repository contains the data and code used in my undergraduate dissertation.

## Data
- `analysis_data.csv`: cleaned dataset of European Parliament speeches on the European Green Deal

## Code
- main scripts for data cleaning, dictionary construction, and analysis

## Results Figures
- "results_section_figures.R": Code to make graphs and tables found in the dissertation

## Methods
The analysis uses dictionary-based text analysis to measure:
- scientific framing
- social framing
- emotive language

## Replication
To replicate the results:
1. Load the dataset
2. Run the script in R
3. Install required packages (quanteda, tidyverse, etc.)
